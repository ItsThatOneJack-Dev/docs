## Userscript (Browser)

The easiest way to use RugplaySDK is as a Tampermonkey userscript on rugplay.com.

[**1. Install Tampermonkey for your browser.**](https://www.tampermonkey.net/)

[**2. Directly install RugplaySDK**](https://raw.githubusercontent.com/ItsThatOneJack-Dev/rugplaysdk/refs/heads/main/RugplaySDK.user.js), or import it via `@require`:

```js
// ==UserScript==
// @name         My Rugplay Bot
// @match        *://*.rugplay.com/*
// @require      https://raw.githubusercontent.com/ItsThatOneJack-Dev/rugplaysdk/refs/heads/main/RugplaySDK.user.js
// @grant        none
// ==/UserScript==

// rugplaySDK and RPSDK$ are now available globally
```

**3. The SDK auto-connects** to Chat and Events when you load rugplay.com while logged in. No setup needed.

<Tabs>
  <Tab title="TypeScript">
    ```ts
    const me = await rugplaySDK.User.me();
    console.log(`Logged in as ${me.displayName}`);
    
    const coin = await rugplaySDK.Coin("CATBOI");
    await coin.buy(100);
    ```
  </Tab>
  <Tab title="JavaScript">

  </Tab>

## Server Script (Node / Bun / Deno)

For running bots outside the browser you need a session cookie from rugplay.com.

**1. Grab your session cookies** from rugplay.com using your browser's DevTools (Application → Cookies):

- `session_data`
- `session_token`
- `cf_clearance`

**2. Install the package:**

```bash
# Bun
bun add rugplaysdk

# npm
npm install rugplaysdk
```

**3. Write your script:**

<Tabs>
  <Tab title="TypeScript">
    ```ts
    import { rugplaySDK, ServerAuth, shutdown } from "rugplaysdk";
    
    ServerAuth.init({
        sessionData:  "eyJ...",
        sessionToken: decodeURIComponent("s%3A..."),
        cfClearance:  "abc123...",
    });
    
    if (!await ServerAuth.validate()) {
        console.error("Invalid or expired session.");
        process.exit(1);
    }
    
    await rugplaySDK.Chat.connect();
    
    const me = await rugplaySDK.User.me();
    console.log(`Running as ${me.displayName}`);
    
    const coin = await rugplaySDK.Coin("CATBOI");
    await coin.buy(100);
    
    shutdown();
    ```
  </Tab>
  <Tab title="JavaScript">
    ```js
    import { rugplaySDK, ServerAuth, shutdown } from "rugplaysdk";
    
    ServerAuth.init({
        sessionData:  "eyJ...",
        sessionToken: decodeURIComponent("s%3A..."),

    ```
  </Tab>
</Tabs>

<Warning>
  Never commit your session cookies to source control. Use environment variables instead:

  ```ts
  ServerAuth.init({
      sessionData:  process.env.SESSION_DATA!,
      sessionToken: process.env.SESSION_TOKEN!,
      cfClearance:  process.env.CF_CLEARANCE!,
  });
  ```
</Warning>

## Checking Login State

<Tabs>
  <Tab title="TypeScript">
    ```ts
    const loggedIn = await rugplaySDK.isLoggedIn();
    if (!loggedIn) {
        console.log("Not logged in.");
    }
    
    // Or get the full user object:
    const me = await rugplaySDK.User.me(); // null if not logged in
    ```
  </Tab>
  <Tab title="JavaScript">
