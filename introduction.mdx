# RugplaySDK

RugplaySDK is a fully-featured TypeScript SDK for interacting with [Rugplay](https://rugplay.com) — a browser-based crypto trading simulator. It provides a clean, typed API for trading, chatting, gambling, managing your profile, and more.

## Two Distributions

RugplaySDK ships in two forms depending on how you want to use it.

<CardGroup cols={2}>
  <Card icon="browser" title="Userscript">
    **`rugplaySDK.user.js`** — install via Tampermonkey or Greasemonkey. Runs in the browser on rugplay.com, auto-connects to Chat and Events on page load.
  </Card>
  <Card icon="server" title="Server">
    **`rugplaySDK.server.js`** — a Node/Bun/Deno-compatible ESM module for running bots and automation scripts outside the browser.
  </Card>
</CardGroup>

## Features

<AccordionGroup>
  <Accordion title="Trading">
    Buy and sell coins, track your portfolio, get transaction history, and monitor live trades via the Events WebSocket.
  </Accordion>
  <Accordion title="Chat & Messaging">
    Send messages, create group channels, send Rugpay requests, and listen to incoming messages in real time via the Chat WebSocket.
  </Accordion>
  <Accordion title="Arcade">
    Play Coinflip, Dice, Slots, Mines, and Tower with full async game state management, auto-cashout timers, and percent-based betting.
  </Accordion>
  <Accordion title="Shop & Inventory">
    Open crates, equip name styles, check gem balances, and manage your inventory.
  </Accordion>
  <Accordion title="Hopium">
    Browse, create, and bet on Hopium prediction questions.
  </Accordion>
  <Accordion title="Leaderboards & Achievements">
    Fetch live leaderboard data, track achievement progress, and claim rewards.
  </Accordion>
  <Accordion title="Backoff & Utilities">
    Built-in exponential, linear, logarithmic, stepped, and custom backoff helpers. HTML sanitisation, Levenshtein distance, and more.
  </Accordion>
</AccordionGroup>

## Global Aliases

When used as a userscript, the SDK is available on `window` under two aliases:

```js
window.rugplaySDK  // full name
window.RPSDK$      // short alias
```
